Design language specification: palette, typography, spacing, and component tokens

Philosophy

Warm restraint

Off-white surfaces, never pure white. The page breathes.

Data-first clarity

Numbers lead. Decoration earns its place.

Editorial weight

Type does the heavy lifting. Hierarchy through size and weight, not color.

Purposeful color

Color signals state, not decoration. One accent. Amber for urgency only.

Component tokens

Card

bg white, border 0.5px default, radius 12px, padding 16px 20px. No shadows.

Page background

#FAF9F6. Cards sit on this. Never pure white for the canvas.

Input

bg white, border 0.5px default, radius 8px, height 36px, 14px text. Focus: border strong + 2px coral offset ring.

Primary button

bg coral-600, text white, radius 8px, height 36px, 13px/500. Hover: coral-700. No shadows.

Secondary button

bg transparent, border 0.5px strong, text primary, radius 8px. Hover: bg subtle.

Sidebar nav item

14px, text secondary. Active: bg subtle, text primary, left border 2px coral-600, radius 6px. No icon color changes.

Status badge

Pill shape, 11px/500. Urgent: amber-50 bg + amber-800 text. Overdue: red-50 + red-800. Paid: green-50 + green-800.

List row

Left border 1.5px border-default (no radius), 15px title/500, 12px meta/tertiary, amount right-aligned 15px/500. Hover: bg subtle.

Interaction principles

Transitions

150ms ease for color/border changes. 200ms ease-out for layout shifts. No spring physics.

Focus

2px coral-600 offset ring. Never suppress :focus-visible. Keyboard-first.

Elevation

No drop shadows. Elevation is expressed through border weight and background contrast only.

Motion

Respect prefers-reduced-motion. Animations are optional polish, never load-bearing.

What this system is not

No gradients on surfaces or buttons. No drop shadows anywhere. No dark glass / frosted effects. No bright blue as primary accent. No colorful sidebar backgrounds. No decorative icons in headers. No more than two typefaces. No using color to create hierarchy between body text sizes.
